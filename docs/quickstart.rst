.. _quickstart:

===========
Quick start
===========

The best start for *BathyPy* is to go through a number of examples that demonstrate how to obtain bathymetry and hypsometry
from our methods. All methods behind the analyses are documented in our :ref:`API section <api>`

.. _examples:

List of examples
================

For a static (non-interactive) view of the examples follow one of the links below:

* `Deriving bathymetry and hypsometry from a static topography dataset <_examples/01_derive_bathymetry_from_static_topography_dataset.ipynb>`_
* `Deriving bathymetry and hypsometry from dynamic ICESat-2 altimetry and Landsat- and Sentinel-2-based surface area estimates <_examples/02_derive_dynamic_hypso.ipynb>`_
* `Combining hypsometry from static topography and dynamic ICESat-2 datasets <_examples/03_combine_static_dynamic_hypso.ipynb>`_
